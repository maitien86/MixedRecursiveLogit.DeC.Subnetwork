------------------------------------------------------------
Mixed Recursive Logit with Error Component model estimation

Mai, T., Bastin, F., and Frejinger, E. A decomposition method for estimating recursive logit
based route choice models. CIRRELT Technical Report, No 66, 2015a.
------------------------------------------------------------
+ Mixed Recursive Logit with Error Component (based on subnetwork components)
+ Borlange network, simulated data
+ Decomposition method
+ Common random numbers for simulating path choice probs
+ Global Link Size attribute
+ Basic trust region algorithms
+ Line search algorithms
+ Hessian approximation: BHHH, BFGS, SR1, CB_SR1, CB_BFGS, BIGGS.
+ Variance Covariance matrix.
